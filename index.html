Code Rush

Project Brief:

I enjoy playing games and I always wanted to learn game development. I came up with an idea for my college project and the title is "CODE RUSH". It is an endless runner game build for Android platform and its all about learn with fun.


Unlike other familiar games, there will not be plain coins instead the coins have the operators and the keywords with the logos of the programming languages. The theme is Java vs Python. The player is free to collect keywords of Java but the keywords of Python are the obstacles. The aim of my project is to attract children and beginners towards coding. Also I wanted to make the player familiar with the keywords. I also added short snippets to the obstacles which direct the player.


I had fun and enjoyed using Blender for making 3d objects, the prefabs. I also used Microsoft Paint for the textures. Arranging the player, the coins, the obstacles in Unity, making animations, applying physics and creating scenes is more fun. For player animations I used Mixamo. I used C# for scripts. I also used Android Studio to build my game.

Project Info

  • Date - May 2021 - June 2021
  • Tools - Unity, C Sharp, Blender, Mixamo, Android Studio
